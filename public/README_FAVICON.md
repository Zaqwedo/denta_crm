# Favicon файлы

Поместите сюда ваши иконки:

1. **icon.png** - основная иконка (32x32 или 64x64 пикселей)
   - Будет отображаться на вкладке браузера
   - Рекомендуемый размер: 64x64 для лучшего качества

2. **apple-icon.png** - иконка для iOS устройств (180x180 пикселей)
   - Используется при добавлении на главный экран iPhone/iPad

## Как добавить:

1. Сохраните ваше изображение с логотипом (зуб с буквами BK)
2. Измените размер до нужных размеров (можно использовать любой редактор изображений)
3. Сохраните как:
   - `icon.png` (64x64)
   - `apple-icon.png` (180x180)
4. Поместите оба файла в эту директорию (`public/`)
5. Перезапустите dev-сервер

После этого иконка появится на вкладке браузера!
