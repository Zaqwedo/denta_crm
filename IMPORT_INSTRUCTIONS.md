# Инструкция по импорту данных из Glide_KVA.csv

## Шаг 1: Изменение типа колонки "Дата рождения пациента" на DATE

1. Откройте Supabase Dashboard
2. Перейдите в SQL Editor
3. Выполните скрипт из файла `supabase-change-birthdate-to-date.sql`

Этот скрипт:
- Конвертирует существующие даты из DD.MM.YYYY в YYYY-MM-DD
- Изменяет тип колонки с TEXT на DATE

## Шаг 2: Импорт данных

После выполнения SQL скрипта, запустите:

```bash
node import-glide-kva.js
```

Скрипт автоматически:
- Читает файл Glide_KVA.csv
- Конвертирует все даты рождения из DD.MM.YYYY в YYYY-MM-DD
- Импортирует все записи в базу данных

## Формат дат

- **Входной формат**: DD.MM.YYYY (например, 21.01.1992)
- **Выходной формат**: YYYY-MM-DD (например, 1992-01-21) - ISO формат для DATE типа
